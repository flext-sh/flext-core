## 0.12.0 (2026-03-05)

**Tag**: `v0.12.0`
**Package**: `flext-core==0.12.0`

### Highlights

- Stable release of `flext-core` — the foundational library for the FLEXT ecosystem.
- Railway-Oriented Programming, Dependency Injection, and CQRS Dispatcher are fully supported and type-safe.
- All architectural primitives (`FlextResult`, `FlextContainer`, `FlextDispatcher`, `FlextModels`, `FlextService`) are production-ready.

### Installation

```bash
pip install flext-core==0.12.0
```

Or with Poetry:

```bash
poetry add flext-core==0.12.0
```

### What's Changed

- Version bumped from `0.10.0-dev` to `0.12.0`.
- Updated dev dependency: `autoflake` 2.3.2 → 2.3.3.
- Documentation links verified; contributing guide available at `docs/development/contributing.md`.
